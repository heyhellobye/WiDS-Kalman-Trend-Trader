# WiDS Kalman Filtered Trend Trader

This repository contains weekly assignments and the final project completed as part of the
WiDS Kalman Filtered Trend Trader program.

## Structure
- Week 1: Regression, bias analysis, and neural networks
- Week 2: Kalman filter based trading strategy
- Week 3: Reinforcement learning (Q-learning and DQN)
- Final Project: Multi-asset Kalman-filtered trend-following strategy
